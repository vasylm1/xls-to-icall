\How to Use

Prepare your Excel file (.xls or .xlsx) with the following structure:

Start DateEnd DateTitleDescriptionRoomLocation
05.04.2025 09:0005.04.2025 10:30Project KickoffInitial MeetingConf Room AHQ - Warsaw

Notes:

  • Start Date and End Date must be in format dd.mm.yyyy hh:mm
  • Title is optional (defaults to "Class")
  • Description and Room are optional (joined together)
  • Location is optional (defaults to "Online")
  • Only the first sheet of your Excel file is used
